Installation Steps

  1. Extract the downloaded ZIP file to your XAMPP htdocs directory:
    C:\xampp\htdocs\smart-restaurant
  2. Start XAMPP:
    • Start Apache
    • Start MySQL
  3. Set up the database:
    • Open phpMyAdmin: http://localhost/phpmyadmin
    • Create new database: smart_restaurant
    • Import backend/db/db_schema.sql
  4. Configure database connection:
    Edit backend/config/config.php: 
    define('DB_HOST', 'localhost');
define('DB_USER', 'root');  // default XAMPP username
define('DB_PASS', '');      // default XAMPP password
define('DB_NAME', 'smart_restaurant');
  5. Access the system:
    • Main Website: http://localhost/smart-restaurant
    • Admin Panel: http://localhost/smart-restaurant/pages/admin/dashboard.html

System Requirements

  • XAMPP (includes Apache, PHP, MySQL)
  • PHP 7.4 or higher
  • MySQL 5.7 or higher
  • Web Browser (Chrome, Firefox, Safari)

Troubleshooting

  • If you get a database connection error:
    • Verify MySQL is running in XAMPP
    • Check database credentials in config.php
    • Ensure database 'smart_restaurant' exists
  • If pages don't load properly:
    • Verify Apache is running in XAMPP
    • Check file permissions
    • Clear browser cache
